# Roberto Beardo García — Portfolio

Portfolio personal desplegado como sitio estático con GitHub Pages.

## Despliegue en GitHub Pages

1. Crea un repositorio en GitHub con el nombre `tuusername.github.io`
2. Sube todos los archivos de este proyecto al repositorio
3. Ve a **Settings > Pages**
4. En **Source** selecciona la rama `main` y la carpeta `/ (root)`
5. Accede a `https://tuusername.github.io`

## Personalizar

| Elemento | Archivo | Buscar |
|----------|---------|--------|
| Nombre y descripcion | `index.html` | `<!-- EDITAR -->` |
| Foto de perfil | `assets/img/avatar.png` | Reemplazar archivo |
| CV en PDF | `assets/cv/CV_Roberto_Beardo.pdf` | Reemplazar archivo |
| Links sociales | `index.html` | `<!-- EDITAR: Pon tu URL -->` |
| Proyectos | `index.html` | `<!-- AÑADIR PROYECTO -->` |
| Colores | `style.css` | `:root` variables |
| Textos del typing | `script.js` | `const strings` |

## Tecnologias

- Google Fonts (JetBrains Mono + Nunito)
